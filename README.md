我需要开发一款在线PDF文档阅读器，要求完美兼容电脑端与移动端，提供管理员界面与普通用户浏览界面。

**用户角色与权限：**
- 管理员分为两种：
  1. 超级管理员：拥有系统最高权限。
  2. 注册管理员：通过注册流程获得管理员身份，具备上传与分享PDF的权限。

**核心功能：**
1. **管理员功能**：
   - 上传PDF文档。
   - 分享PDF，支持以下两种模式：
     a) **试读模式**：可设置允许试读的页数（如前几页）。
     b) **浏览模式**：需填写接收者的姓名与邮箱。接收者通过邮件认证后方可访问PDF。

2. **普通用户访问控制**：
   - **浏览模式**：
     - 用户通过分享链接进入，需完成邮件认证。
     - 阅读界面为全屏模式，并添加全屏的半透明水印，水印内容包含用户姓名和提示文字。
     - **严格禁止下载、打印或复制PDF内容**。
   - **试读模式**：
     - 用户可直接查看PDF的前N页（页数由管理员设定），无需认证。
     - 同样禁止下载与复制。

**其他要求：**
- 系统需具备响应式设计，确保在电脑、手机等设备上均有良好体验。
- 管理员界面应支持文档管理、分享记录查看等功能。
- 普通用户界面应简洁易用，阅读时支持放大、缩小、翻页等基本操作。

请根据以上需求进行系统设计与开发，并确保安全性与用户体验。
